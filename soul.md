## 第一阶段：原始数据采集 (Raw Scoring)

设置四个维度变量，每个维度的初始分值为 0。
* $P$ (Participation): 互动维度
* $V$ (Values): 内核维度
* $I$ (Information): 表达维度
* $E$ (Emotion): 生存维度

对于每一组的 8 道题（共 32 题）：
* 选择 **A**：该维度 $n_A + 1$
* 选择 **B**：该维度 $n_B + 1$
* 选择 **C**：该维度 $n_C + 1$（不计入 A/B 权重，但会稀释强度，模拟“中立/摇摆”）

**维度系数 $\beta$ 计算公式：**
为了后续计算，我们需要将每个维度的得分转化为 $[0, 1]$ 之间的连续变量 $\beta$。
$$\beta_d = \frac{(n_{A,d} - n_{B,d}) + 8}{16}$$
*(注：当 $\beta > 0.5$ 偏向 A 属性，$\beta < 0.5$ 偏向 B 属性，$\beta = 0.5$ 为绝对中立。)*

## 第二阶段：核心属性判定 (The 4-Letter Code)

这是决定用户最终“人格代码”的逻辑，采用**“抽象优先原则”**处理平局（Deadlock）。

| 维度 | 判定逻辑 | 结果代码 |
| :--- | :--- | :--- |
| **互动 (P)** | $IF \ n_{A,P} > n_{B,P} \ THEN \ \mathbf{E} \ ELSE \ \mathbf{I}$ | **E**xport / **I**nternal |
| **内核 (V)** | $IF \ n_{A,V} > n_{B,V} \ THEN \ \mathbf{S} \ ELSE \ \mathbf{D}$ | **S**incere / **D**econstruct |
| **表达 (I)** | $IF \ n_{A,I} > n_{B,I} \ THEN \ \mathbf{R} \ ELSE \ \mathbf{M}$ | **R**ational / **M**eme |
| **生存 (E)** | $IF \ n_{A,E} > n_{B,E} \ THEN \ \mathbf{A} \ ELSE \ \mathbf{B}$ | **A**ttack / **B**lock |

**无死角处理：** 若 $n_A = n_B$，则系统强制赋予“抽象底色字母”（即 **I, D, M, B**），这保证了程序的唯一输出。

## 第三阶段：16维指纹映射公式 (Fingerprint Matrix)

这是让用户觉得“准”的核心。我们不直接用原始分，而是通过线性加权公式，计算出 16 个二级指标（分值范围 0-100）。

我们将维度系数简写为：$\beta_P, \beta_V, \beta_I, \beta_E$。

### 1. 精神内核模型 (Core)
* **C1 自我确信度：** $(\beta_E \times 0.6 + \beta_P \times 0.4) \times 100$
* **C2 意义渴求度：** $(\beta_V \times 0.8 + \beta_I \times 0.2) \times 100$
* **C3 道德敏感感：** $(\beta_V \times 0.5 + \beta_R \times 0.5) \times 100$
* **C4 虚无主义边界：** $((1 - \beta_V) \times 0.9 + (1 - \beta_P) \times 0.1) \times 100$

### 2. 情感与社交模型 (Social)
* **S1 社交表达欲：** $\beta_P \times 100$
* **S2 情感剥离度：** $((1 - \beta_V) \times 0.7 + \beta_P \times 0.3) \times 100$
* **S3 人际结界厚度：** $((1 - \beta_P) \times 0.8 + (1 - \beta_E) \times 0.2) \times 100$
* **S4 赛博共情力：** $(\beta_V \times 0.6 + (1 - \beta_E) \times 0.4) \times 100$

### 3. 认知与信息模型 (Cognition)
* **I1 逻辑强迫症：** $\beta_I \times 0.9 \times 100$
* **I2 抽象感染力：** $((1 - \beta_I) \times 0.7 + (1 - \beta_V) \times 0.3) \times 100$
* **I3 观点固化度：** $(\beta_E \times 0.5 + \beta_I \times 0.5) \times 100$
* **I4 伪装熟练度：** $((1 - \beta_V) \times 0.5 + (1 - \beta_E) \times 0.5) \times 100$

### 4. 行为与生存模型 (Action)
* **A1 赛博攻击性：** $\beta_E \times 100$
* **A2 情绪自愈力：** $((1 - \beta_V) \times 0.6 + (1 - \beta_E) \times 0.4) \times 100$
* **A3 秩序破坏欲：** $((1 - \beta_V) \times 0.4 + \beta_E \times 0.6) \times 100$
* **A4 摆烂执行力：** $(1 - \beta_E) \times 100$

## 第四阶段：牛马指数 (The "Cattle" Index) 计算

牛马指数不是随机的，它是基于“防御性”和“真诚度”的加权结果。
**公式：**
$$Index_{NM} = \left[ (1 - \beta_E) \times 0.5 + (1 - \beta_V) \times 0.3 + (1 - \beta_P) \times 0.2 \right] \times 100\%$$
* **逻辑：** 越是不敢反抗（低 $\beta_E$）、越是看透世界（低 $\beta_V$）、越是潜水（低 $\beta_P$），牛马纯度越高。

## 第五阶段：扎心画像生成逻辑 (The Roast Algorithm)

为了做到“直击灵魂”，你需要为 16 个子维度设定阈值触发器。

* **触发器 A (高敏型)：** IF $S4 > 85$ AND $I4 < 20$ → 插入文案：“你不仅被生活磨平了棱角，甚至还在帮生活数钱。”
* **触发器 B (防卫型)：** IF $A4 > 90$ AND $C1 < 30$ → 插入文案：“你所谓的摆烂其实是由于太害怕失败，所以你干脆连入场的机会都不给自己。”
* **触发器 C (虚无型)：** IF $C4 > 80$ AND $S2 > 80$ → 插入文案：“你看着网上的人群吵架就像看猴子杂耍，虽然你在笑，但你其实觉得这些笑声也毫无意义。”

## 第一阶段：原始数据采集 (Raw Scoring)

设置四个维度变量，每个维度的初始分值为 0。
* $P$ (Participation): 互动维度
* $V$ (Values): 内核维度
* $I$ (Information): 表达维度
* $E$ (Emotion): 生存维度

对于每一组的 8 道题（共 32 题）：
* 选择 **A**：该维度 $n_A + 1$
* 选择 **B**：该维度 $n_B + 1$
* 选择 **C**：该维度 $n_C + 1$（不计入 A/B 权重，但会稀释强度，模拟“中立/摇摆”）

**维度系数 $\beta$ 计算公式：**
为了后续计算，我们需要将每个维度的得分转化为 $[0, 1]$ 之间的连续变量 $\beta$。
$$\beta_d = \frac{(n_{A,d} - n_{B,d}) + 8}{16}$$
*(注：当 $\beta > 0.5$ 偏向 A 属性，$\beta < 0.5$ 偏向 B 属性，$\beta = 0.5$ 为绝对中立。)*

## 第二阶段：核心属性判定 (The 4-Letter Code)

这是决定用户最终“人格代码”的逻辑，采用**“抽象优先原则”**处理平局（Deadlock）。

| 维度 | 判定逻辑 | 结果代码 |
| :--- | :--- | :--- |
| **互动 (P)** | $IF \ n_{A,P} > n_{B,P} \ THEN \ \mathbf{E} \ ELSE \ \mathbf{I}$ | **E**xport / **I**nternal |
| **内核 (V)** | $IF \ n_{A,V} > n_{B,V} \ THEN \ \mathbf{S} \ ELSE \ \mathbf{D}$ | **S**incere / **D**econstruct |
| **表达 (I)** | $IF \ n_{A,I} > n_{B,I} \ THEN \ \mathbf{R} \ ELSE \ \mathbf{M}$ | **R**ational / **M**eme |
| **生存 (E)** | $IF \ n_{A,E} > n_{B,E} \ THEN \ \mathbf{A} \ ELSE \ \mathbf{B}$ | **A**ttack / **B**lock |

**无死角处理：** 若 $n_A = n_B$，则系统强制赋予“抽象底色字母”（即 **I, D, M, B**），这保证了程序的唯一输出。

## 第三阶段：16维指纹映射公式 (Fingerprint Matrix)

这是让用户觉得“准”的核心。我们不直接用原始分，而是通过线性加权公式，计算出 16 个二级指标（分值范围 0-100）。

我们将维度系数简写为：$\beta_P, \beta_V, \beta_I, \beta_E$。

### 1. 精神内核模型 (Core)
* **C1 自我确信度：** $(\beta_E \times 0.6 + \beta_P \times 0.4) \times 100$
* **C2 意义渴求度：** $(\beta_V \times 0.8 + \beta_I \times 0.2) \times 100$
* **C3 道德敏感感：** $(\beta_V \times 0.5 + \beta_R \times 0.5) \times 100$
* **C4 虚无主义边界：** $((1 - \beta_V) \times 0.9 + (1 - \beta_P) \times 0.1) \times 100$

### 2. 情感与社交模型 (Social)
* **S1 社交表达欲：** $\beta_P \times 100$
* **S2 情感剥离度：** $((1 - \beta_V) \times 0.7 + \beta_P \times 0.3) \times 100$
* **S3 人际结界厚度：** $((1 - \beta_P) \times 0.8 + (1 - \beta_E) \times 0.2) \times 100$
* **S4 赛博共情力：** $(\beta_V \times 0.6 + (1 - \beta_E) \times 0.4) \times 100$

### 3. 认知与信息模型 (Cognition)
* **I1 逻辑强迫症：** $\beta_I \times 0.9 \times 100$
* **I2 抽象感染力：** $((1 - \beta_I) \times 0.7 + (1 - \beta_V) \times 0.3) \times 100$
* **I3 观点固化度：** $(\beta_E \times 0.5 + \beta_I \times 0.5) \times 100$
* **I4 伪装熟练度：** $((1 - \beta_V) \times 0.5 + (1 - \beta_E) \times 0.5) \times 100$

### 4. 行为与生存模型 (Action)
* **A1 赛博攻击性：** $\beta_E \times 100$
* **A2 情绪自愈力：** $((1 - \beta_V) \times 0.6 + (1 - \beta_E) \times 0.4) \times 100$
* **A3 秩序破坏欲：** $((1 - \beta_V) \times 0.4 + \beta_E \times 0.6) \times 100$
* **A4 摆烂执行力：** $(1 - \beta_E) \times 100$

## 第四阶段：牛马指数 (The "Cattle" Index) 计算

牛马指数不是随机的，它是基于“防御性”和“真诚度”的加权结果。
**公式：**
$$Index_{NM} = \left[ (1 - \beta_E) \times 0.5 + (1 - \beta_V) \times 0.3 + (1 - \beta_P) \times 0.2 \right] \times 100\%$$
* **逻辑：** 越是不敢反抗（低 $\beta_E$）、越是看透世界（低 $\beta_V$）、越是潜水（低 $\beta_P$），牛马纯度越高。
